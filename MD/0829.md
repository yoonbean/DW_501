### 1. Javascript for문
#### 1. for 문이란?
    주어진 조건에 따라서 반복해서 실행되는 문법.
    활용 범위가 넓은 문법이며 사용 빈도가 높다.

#### 2. for 문 해석
       
```javascript     
       for(var i=0; i<3; i++){
            console.log(i);
        }    

// 1.  var i=0;              초기문
// 2.  i<3;                  조건식
// 3.  console.log(i);       작업문
// 4.  i++ (++는 1씩 증가)   반복 후 작업
// 초기문 -> 조건식 -> 작업문 -> 증가식 순으로 진행.
```         

#### 3. for 문 예제

```javascript

            if(i%2 == 0){
            console.log("i는 : " + i + "짝수입니다.");
            }

    // 데이터 타입 var 인 변수 i 에 1을 대입.

    // i = 1 이면 결과 출력 x

    // i = 2 이면 i는 2 짝수입니다. 출력
    // i = 3 이면 결과 출력 x
    // i = 4 이면 i는 4 짝수입니다. 출력

    // if 문은 i 가 4 와 같다면 i는 4입니다. 출력.


         for(var i=0; i<=5; i++){
            if(i == 2 || i == 4){

    // 데이터 타입 var 인 변수 i 에 0을 대입.

        var count = 0;
        for(var i=0; i<=10; i++){
            if(i%2 == 0){
               count++;
            }
        }
        console.log("짝수 개수는" + count + "개 입니다.")

        // 데이터 타입이 var 인 count 변수에 0을 대입하여 변수 선언.
        // 데이터 타입이 var 인 변수 i에 0을 대입.
        // i 가 10보다 작거나 같다면 if 문 실행.

        var sum = 0; 

        for(var i=1; i<=10; i++){
    
            sum=sum+i;

        }
        console.log(sum);

        // 데이터 타입이 var 인 변수 sum에 0을 대입하여 변수 선언.

        // i=1 일 때 sum= 0 + 1 
        // console.log(sum);은 1이 된다.

        // i=2 가 된다. 이 때 다시 sum=sum+1 이 실행되므로, sum= 1+1 즉, 2가 된다.
        // 이렇게 쭉 가다보면
        // i=10 일 때 sum=sum+1 은 45+10이므로 55가 된다.
```
